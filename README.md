# 🎵 Wavelength

**Wavelength** is a modern Spotify-inspired music streaming web application built using **HTML, CSS, and JavaScript**. It provides an elegant and responsive interface for playing locally stored music, organizing songs into playlists, searching tracks, and managing favorites—all while delivering a user experience inspired by Spotify.

---


## ✨ Features

### 🎵 Music Playback
- Play, pause, next, and previous controls
- Interactive progress bar
- Volume control
- Display of current song information

### 📂 Playlist Organization
- Browse songs by curated playlists
- Categories including:
  - ❤️ Favorite Songs
  - 🌙 Soft & Melodious
  - 🇵🇰 Urdu & Indian Hits
  - 🌍 English Essentials

### 🔍 Search
- Search songs by title or artist
- Instant filtering of available tracks

### 🎨 Modern Spotify-Inspired UI
- Dark theme interface
- Responsive sidebar navigation
- Interactive song cards
- Featured playlist section
- Bottom music player

### 💻 Responsive Design
- Optimized layout for desktop
- Flexible grid-based song display
- Smooth user interactions and hover animations

---

## 🛠️ Tech Stack

- **HTML5** – Semantic page structure
- **CSS3** – Styling, layouts, animations, and responsive design
- **JavaScript (ES6)** – Music playback, search functionality, playlist management, and UI interactions
